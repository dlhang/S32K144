# S32K144 DHT11 Sensor Interface

This project demonstrates how to interface the **DHT11 temperature and humidity sensor** with the **NXP S32K144 microcontroller**.

## Hardware

- NXP S32K144 EVB
- DHT11 Temperature & Humidity Sensor

## Features

- Bit-banged DHT11 communication
- Temperature and humidity reading
- UART debug output

## Project Structure
```
DHT11/
├── src
├── board

```
