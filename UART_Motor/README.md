# S32K144 UART Motor Control

This project demonstrates UART communication to control a motor using the NXP S32K144 microcontroller.

## Hardware
- NXP S32K144 EVB
- Motor driver (L298N / similar)
- UART connection

## Features
- UART command interface
- Motor forward / backward
- Stop control

## Project Structure

UART_Motor/
├── src
├── board
├── Generated_Code
└── Project_Settings

## Build

Open the project with **S32 Design Studio** and build normally.
